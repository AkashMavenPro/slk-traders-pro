# SLK Traders Pro — Automated Agent Configuration

## Agent Skills

### Research Agent
- **Capability**: Web scraping and data extraction from LinkedIn and IndiaMart
- **Use Cases**: Lead generation, supplier research, competitor analysis, contact enrichment
- **Tools**: Puppeteer/Playwright for scraping, data normalization pipelines

### Architect Agent
- **Capability**: Design and scaffold React/Next.js project structures
- **Use Cases**: Page routing, component hierarchy, API route design, folder conventions, SSR/SSG strategy
- **Stack**: Next.js (App Router), TypeScript, Tailwind CSS, modular feature-based architecture

### UI/UX Agent
- **Capability**: Industrial Dark Mode UI with Professional Safety Orange branding
- **Design System**:
  - Primary: Safety Orange `#FF6B00`
  - Background: Dark Charcoal `#1A1A1A`
  - Surface: Dark Gray `#2C2C2C`
  - Text: Off-White `#F5F5F5`
  - Accent: Amber `#FFA500`
- **Use Cases**: Component design, layout, theming, accessibility, responsive design

### Database Agent
- **Capability**: Supabase backend setup and schema design
- **Use Cases**: Table design, Row-Level Security (RLS) policies, Auth configuration, real-time subscriptions, storage buckets, Edge Functions
- **Stack**: Supabase (PostgreSQL), Prisma or Supabase client, REST/GraphQL API

## Project Context
- **Project**: SLK Traders Pro
- **Platform**: Windows 11, Node.js/Next.js
- **Goal**: Professional industrial trade platform with dark-mode branding
